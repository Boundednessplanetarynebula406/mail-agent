# Support

## Where to ask for help

- usage questions: open a GitHub discussion or issue when available
- bug reports: open an issue with reproduction steps
- security issues: follow [SECURITY.md](./SECURITY.md)

## Include this context

- OS
- Node version
- whether you installed from source or npm
- whether the secret backend was `keytar` or `file`
- whether the problem is mail, calendar, contacts, plugin install, or packaging

## Please do not post

- live tokens
- app passwords
- raw mailbox contents
- exported contact books
